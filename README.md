# HealthMetrics
HealthMetrics is a web-based data analytics project that tracks and visualizes daily fitness metrics such as steps, calories, and sleep. Built using HTML, CSS, and JavaScript with Chart.js, it provides insightful health trends, weekly progress charts, and personalized recommendations to help users improve their overall fitness and lifestyle.
